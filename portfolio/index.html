Project Auguste

Project Auguste, a touch-first interactive sculpting mobile app, is part of Adobe's Project Felix initiative.

Goal:
My task was to take technology and wrap a product design around it that would comply with and extend Adobe's application aesthetic. I Was given four months to produce a demo app to be presented at Adobe MAX 2016.

Process:
Project Auguste's engine was designed by Adobe's Sebastian Marketsmueller and allows the user to sculpt in a virtual space, complete with various brushes, filters, and symmetrical drawing effects. I introduced an interaction system that would allow for users to interact with the camera object while at the same time editing the object's surface using a combination of multi-touch gestures and optionally using a stylus.

Outcome:
We produced an Windows SurfaceBook app that aligned perfectly with Adobe's world-class products, while extending the current input conventions allowing for a rich multitouch and touch-stylus gesture system.

Conclusion:
Althought the product was well received, the community feedback was:

  • Creative professionals want to use tools like Z-Brush
  • The target market for the product was more artist/sculptural, as opposed to creative hobiest.
  • 3D is still a touch sell for creatives and non-creatives alike and this was too niche.

CraftMaster Software

CraftMaster Software was started with a singular purpose, to create a tool for users of all ages to import, export, beautify, and share there MineCraft creations..

Goal:
There were three main goals to address:

  • Create brand, web site, and marketing content.
  • Design a multi-platform application for users of all ages to import/export 3D object to and from Minecraft.

Process:
I worked closely with the team to coordinate design, marketing, and engineering efforts. We used weekly meetings and design sprints to understand the main goals. I created corporte logos, branding guidelines, icons and additional assets for web, marketing, and application use. I worked with Andy to understand the user base and market, then created personals and workflows. Working directly with Pete we quickly iterated thought a tech demo to product demo and started user testing and feedback. We iterated over the software developing the feature set for the final product.

Outcome:
CraftMaster not only allowed for 3D model import/export, but also included a several 3D wherehouses acting a 3D model library. CraftMaster also had the ability to share your 3D models to Facebook and other sites supporting the Shapeways 3D Viewer.

Conclusion:
Although CraftMaster has great potential, due to patent licencing issues the road to profitability was a dead-end and the project had to be canceled. However, the CraftMaster app is still being used by more than 1,000 users.
