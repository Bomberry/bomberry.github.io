HUGO Table of Contents

Working with Hugo’s TableOfContents module

Hugo does a great many things for you, with built-in modules, functions, templates, and shortcodes. One of those usefule elements is the {{ .TableOfContents }} module.

However, the output of the code can be less than beautiful, and a little problematic.

So, how do you fix the output to do things like, add a class, or onclick attribute? JavaScript or JQuery can easily help with this, and I’ll show you how it helped me.

$(document).ready(function(){
    // $("#toc_aside").find("ul, li").contents().unwrap();
    $("#TableOfContents").find("a").attr('data-value', 'no-show');
    $("#TableOfContents").find("a").attr('onclick', 'closeAside()');
});
